# Skeleton Plugin

A template for creating a new plugin for the [ITSM Core](https://github.com/OTRS/itsm).

## Installation

1. Install the plugin as usual.
2. run
```bash
./tools/skeleton.sh
```
with the name of your plugin
3. Adapt the plugin skeleton to your needs.

## Features
- A profile file
- A config route
- translations
